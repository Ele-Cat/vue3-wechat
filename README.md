# Vue3 WeChat

- 一个使用 Vite + Vue3 + AntDesignVue + Pinia + MockJS 搭建的还原新版微信的小项目
- [查看演示](https://ele-cat.gitee.io/vue3-wechat/)

- [x] 界面主体

- [x] 用户列表

- [x] 聊天窗口

## 使用

### 安装

```
npm install
```

### 运行

```
yarn serve
```

### 打包

```
yarn build
```
